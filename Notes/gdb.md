gcc -g ; отладокчные символ

- `count`  — сколько элементов вывести (по умолчанию 1)  
- `size`   — размер каждого элемента  
- `format` — как отображать элемент  
- `<address>` — адрес (можно использовать `$rax`, `($rcx+0x10)` и т.п.)

---

## Размер элемента (`size`)

| Буква | Размер | Описание      |
|-------|--------|---------------|
| `b`   | 1 байт | byte          |
| `h`   | 2 байта| halfword      |
| `w`   | 4 байта| word (32-бит) |
| `g`   | 8 байт | giant (64-бит)|

---

## Формат отображения (`format`)

| Буква | Формат     | Описание                           |
|-------|------------|------------------------------------|
| `x`   | hex        | шестнадцатеричный                  |
| `d`   | signed dec | десятичный (со знаком)             |
| `u`   | unsigned   | десятичный (без знака)             |
| `o`   | octal      | восьмеричный                       |
| `t`   | binary     | двоичный                           |
| `f`   | float      | число с плавающей точкой           |
| `a`   | address    | как адрес/метка                    |
| `i`   | instruction| дизассемблировать как инструкцию   |
| `c`   | char       | символ                             |
| `s`   | string     | строка (C-style, до `\0`)          |

---

## Примеры

```gdb
x/16bx $rcx        # 16 байт по адресу в rcx, в hex
x/gx ($rcx+0x10)   # 8 байт (uint64) по rcx+0x10
x/4wx $rsp         # 4 слова (4 байта) по rsp
x/s $rax           # строка по адресу в rax
x/8i $pc           # 8 инструкций начиная с pc
p/x $rax           # вывести rax в hex
```

## TUI

```
layout asm      # окно с ассемблером
layout split    # исходный код + ассемблер
layout regs     # окно с регистрами
layout src      # только исходный код
tui enable      # включить TUI
tui disable     # выключить TUI
```
```
Ctrl+L — перерисовать интерфейс
Ctrl+X 1 — оставить одно окно
Ctrl+X 2 — разделить на два окна
Ctrl+X o — переключиться между окнами
```
