* Структура программы :basics:reverse:
** Программа это структурированный контейнер которой состоит из
  - Заголовков
  - Секций
  - Машинного кода
  - Таблиц импорта
  - Ресурсов
** Заголовки
Информация для загрущика о типе файла, адресах, размерах, точках входа
** Секции
- .text содержит исполняемый код программы
- .data содержит инициализированные данные - глобальные переменные
- .rodata содержит строки и константы
- .bss содержит неинициализированные данные, место под рантайм инициализацию
** Алгоритм
- Выделить границы функции
- Определить область выделения памяти
- Переименовать явыне переменные
- Выделить блоки которые возможно прокомментировать


* Полезные заметки :notes:
** ChaCha :crypto:
Элементы:
- key 256 битный ключ
- counter 32 битный счетчик
- nonce 96 битное число
- round обычно 20 или 8

  Алгоритм содержит операции:
  - сложение по модулю 2 ^^ 32
  - побитовое сложение по модулю 2
  - циклические сдвиги в лево
Эти операции состовляют quarterRound
Это операция которая принимает массив из 16 32-разрядных unsigned int и набор 4 32-разрядных чисел
Числа обозначаем a b c d
#+begin_src csharp
static void quarterRound(ref uint[16] x, int a, int b, int c, int d)
{
    x[a] += x[b]; x[d] = rotl32(x[d] ^ x[a], 16);
    x[c] += x[d]; x[b] = rotl32(x[b] ^ x[c], 12);
    x[a] += x[b]; x[d] = rotl32(x[d] ^ x[a],  8);
    x[c] += x[d]; x[b] = rotl32(x[b] ^ x[c],  7);
}

static uint rotl32(uint x, int n)
{
    return x << n | (x >> (-n & 31));
}
#+end_src
