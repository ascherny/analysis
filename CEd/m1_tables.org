** SPR
- Регистр флагов
  EFLAGS/RFLAGS на x86
  PSTATE на ARM
  Каждый его бит содержит значение состояния возникшего в результате выполнения инструкции

- Program Counter
  EIP/RIP на x 86
  PC/R15 на ARM
  PC на RISC-V
  Содержит метку следующей инструкции

- Stack Pointer
  ESP/RSP на x86
  SP/R13 на ARM
  SP на RISC-V
  Содержит адрес стека для ориентации в пространстве стека
