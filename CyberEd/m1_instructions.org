* Notes :asm:
push eax использует несколько базовых операций

Команда PUSH уменьшает значение регистра стека на размер операнда (2 или 4) и копирует содержимое операнда в память по адресу SS:SP

#+begin_src asm
    ;push eax
esp = esp - 4
[esp] = eax
#+end_src

уменьшаем значение стека на столько байт сколько нужно чтобы поместить операнд
 кладем значение операнда в память

 Соответственно pop делает наоборот

 Берем значение из ячейки памяти по адресу, кладем в операнд, увеличиваем значение стека на размер операнда

 В ARM эти инструкции выполняются буквально без сахара push

** SR
- Регистр флагов
  EFLAGS/RFLAGS на x86
  PSTATE на ARM
  Каждый его бит содержит значение состояния возникшего в результате выполнения инструкции

- Program Counter
  EIP/RIP на x 86
  PC/R15 на ARM
  PC на RISC-V
  Содержит метку следующей инструкции

- Stack Pointer
  ESP/RSP на x86
  SP/R13 на ARM
  SP на RISC-V
  Содержит адрес стека для ориентации в пространстве стека
