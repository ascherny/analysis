* Инструкция к инструкциям :asm:basics:

r - операнд может быть регистром
m - адрес в памяти
XX - размер операнда

** Перемещение данных

| Операция            | x86            | ARM                       | RISC-V                         |
| Скопировать r1 → r2 | mov r2, r1     | mov x2, x1 (alias)        | addi x2, x1, 0                 |
| Push register       | push eax       | stp x29, x30, [sp, #-16]! | addi sp, sp, -8``sd x1, 0(sp)  |
| Push multiple       | pusha (legacy) | stp x1, x2, [sp, #-16]!   | Нужно делать руками            |
| Pop register        | pop eax        | ldp x29, x30, [sp], #16   | ld x1, 0(sp) \n addi sp, sp, 8 |
| Pop multiple        | popa (legacy)  | ldp x1, x2, [sp], #16     | Нужно делать руками            |
|                     |                |                           |                                |

** Аримфметические и логические инструкции

| Операция | x86                  | ARM            | RISC-V         | Note                     |

| OR       | mov r3, r2 or r3, r1 | orr x3, x1, x2 | orr x3, x1, x2 | or dest, src             |
| XOR      | xor r3, r1           | eor x3, x1, x2 | xor x3, x1, x2 |                          |
| NOT      |                      |                |                |                          |
|          |                      |                |                |                          |

** Условные переходы

| Описание условия для cmp              | x86       | ARM64 | Условие              | Флаги      |
| Равенство операндов                   | je / jz   | b.eq  | ZF == 1              | ZF         |
| Неравенство операндов                 | jne / jnz | b.ne  | ZF == 0              | ZF         |
| Один операнд больше другого           | jg        | b.gt  | ZF == 0 and SF == OF | ZF, SF, OF |
| Один операнд больше или равен другого | jge       | b.ge  | SF == OF             | SF, OF     |
| Один операнд меньше другого           | jl        | b.lt  | SF != OF             | SF, OF     |
| Один операнд меньше или равен другого | jle       | b.le  | ZF == 1 or SF != OF  | ZF, SF, OF |

* Notes :asm:
 push eax использует несколько базовых операций

Команда PUSH уменьшает значение регистра стека на размер операнда (2 или 4) и копирует содержимое операнда в память по адресу SS:SP

#+begin_src asm
    ;push eax
esp = esp - 4
[esp] = eax
#+end_src

уменьшаем значение стека на столько байт сколько нужно чтобы поместить операнд
 кладем значение операнда в память

 Соответственно pop делает наоборот

 Берем значение из ячейки памяти по адресу, кладем в операнд, увеличиваем значение стека на размер операнда

 В ARM эти инструкции выполняются буквально без сахара push
